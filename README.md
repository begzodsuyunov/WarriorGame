# WarriorGame
Controls:
  hold down the screen - forward movement
  release - smoothly stop, at which point you can make a jump
  tap - jump, at a smooth stop jumps forward, if the sword does not move, just jump in place.

Settings
  Buttons:
  On/Off Sound
  On/Off Vibration
  Back to menu button
